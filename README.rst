Running Locally
----------------

.. code:: console

    $ git clone git@github.com:gtalarico/interactive-elastic-analyzer.git
    $ cd interactive-elastic-analyzer
    $ pipenv install --dev
    $ # or virtualenv + pip install -r requirements
    $ python run.py
